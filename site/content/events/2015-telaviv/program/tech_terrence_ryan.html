--- title: Program layout: event_tlv extension: html dirty: true filter: - erb - markdown --- ## Containing Chaos with Kubernetes This session will begin with a level set. If people don't know Docker, I'll catch them up. If they know Docker, I'll move on to making connected Docker container work together to begin with. Then I'll show how to do the same with Kubernetes. I'll then show off the Replication Controller which is the special sauce of Kubernetes which makes sure that if you say you need 5 containers running the web image, and one goes down another one will spin up. All of that should take a good 45 minutes to show with Q/A ## About the speaker - [Terrence Ryan](https://twitter.com/tpryan) Terry Ryan is a Developer Advocate for the Google Cloud Platform team. He has 15 years of experience working with the web - both front end and back. He is passionate about web standards and wants to bring web developers to the Google Cloud Platform. Before Google, he worked for Adobe and the Wharton School of Business. He also wrote Driving Technical Change for Pragmatic Bookshelf, a book that arms technology professionals with the tools to convince reluctant co-workers to adopt new tools and technology.
